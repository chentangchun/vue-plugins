# vue plugins
一些封装的关于vue的插件

[演示demo](https://github.com/windlany/vue-plugins/blob/master/demo/static/show/all.gif)

## 预览效果
```JavaScript
// clone到本地
git clone https://github.com/windlany/vue-plugins.git 
cd demo 
// 安装依赖
npm install
// 运行
npm run dev
```

## 快速使用 
```JavaScript
// main.js
import plugins from "path/vue-plugins";
Vue.use(plugins); 
```
具体使用请看插件详情页
